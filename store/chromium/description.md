# Chrome Web Store / Edge Add-ons Listing

## Name
Nostr Media Gallery

## Short description (max 132 chars)
Browse images & videos from any Nostr user in a beautiful gallery with lightbox, auto-detected from the page URL.

## Detailed description
Nostr Media Gallery lets you instantly browse the media shared by any Nostr user.

**How to use:**
1. Visit any Nostr profile page (e.g. on nostr.com, primal.net, snort.social)
2. Click the extension icon — the Nostr address is auto-detected from the URL
3. Click "Generate Gallery"
4. A new tab opens with all images and videos in a responsive grid
5. Click any image to open it in a full-screen lightbox with keyboard navigation

**Features:**
- Auto-detects npub/nprofile addresses from the current tab URL
- Queries multiple public Nostr relays in parallel
- Responsive masonry-style grid layout
- Full-screen lightbox for images (Escape / arrow keys)
- Video playback with native controls
- Works entirely in your browser — no accounts, no servers, no tracking

**Privacy:**
No data is collected. All processing happens locally in your browser. No analytics, no telemetry.

https://github.com/delirehberi/nostr-media-gallery
